# BuildMyPC India

AI-powered custom PC builder for India. Get personalized component recommendations, compare prices from top Indian retailers, and build your dream PC.

## Features

- AI-powered component recommendations
- Price comparison from Amazon, Flipkart, MD Computers & more
- Compatibility checking
- Budget optimization
- Performance benchmarks
- Popular pre-configured builds

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Shadcn UI
- Framer Motion

## Deployment

Build the project with `npm run build` and deploy the `dist` folder to any static hosting provider (Vercel, Netlify, Cloudflare Pages, etc.).
